# Q-Learning Snake Game

An Android Snake Game powered by Reinforcement Learning, where an AI agent learns to play autonomously.

![AI Demo](ai_demo.gif)

This project combines the classic Snake Game with a Q-Learning based agent.  
The goal is to let the agent learn how to survive longer, collect food, and improve its decisions through interaction with the environment.

## Features

- Q-Learning based AI agent
- Autonomous Snake gameplay
- Reward-driven learning mechanism
- Android implementation with Kotlin
- Classic game environment for Reinforcement Learning experiments

## Tech Stack

- Kotlin
- Android Studio
- Reinforcement Learning
- Q-Learning

## How It Works

The agent interacts with the Snake Game environment and learns by receiving rewards and penalties based on its actions.  
Over time, it improves its strategy to make better movement decisions and achieve higher scores.
